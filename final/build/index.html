Rules

On hover:
Cursor change to pointer
Bell shakes
Bell gets slightly larger

Off hover:
Cursor changes back to original selector
Bell reverts to original size
Bell stops shaking

On click:
Modal appears under the bell
Buttons and text are right aligned

On modal hover:
Background color changes for selections
Text color changes

Off modal hover:
Cursor changes to original

On modal click:
Bell changes to selection
Sound plays for selection excluding "Personalized"
Modal closes
Bell switches to selected bell
